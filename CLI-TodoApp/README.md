# CLI Todo App

## -add string
    - Add a new todo item

## -del int
    - Delete a todo item

## -edit string
    - Edit a todo item

## -list
    - List all todo items

## -toggle int
    - Toggle a todo item

## Usage

> ./todos -add "Buy groceries"

> ./todos -list

> ./todos -del 1

> ./todos -edit 1 "Buy groceries"

> ./todos -toggle 1
